# PARIVESH 3.0 - Navigation & User Guide

## 🚀 Quick Start

### How to Navigate the Portal

1. **Start at Home Page** (`/`)
   - View the landing page with clearance type cards
   - Click "Track Your Proposal" or navigate to Dashboard

2. **Select Your Role** (`/login`)
   - Choose from 4 role types:
     - **Admin** → `/admin`
     - **Project Proponent** → `/proponent`
     - **Scrutiny Team** → `/scrutiny`
     - **MoM Team** → `/mom`

---

## 📋 Role-Specific Workflows

### 👤 Project Proponent Workflow

**Goal**: Submit and track environmental clearance applications

1. **Access Dashboard** → `/proponent`
   - View active applications
   - Check notifications
   - See application pipeline status

2. **File New Application** → Click "File New Application" → `/proponent/new`
   
3. **Complete 5 Steps**:
   - **Step 1**: Select category (A/B1/B2) and sector
   - **Step 2**: Enter project details (name, cost, location, GPS)
   - **Step 3**: Upload required documents
   - **Step 4**: Complete payment via UPI/QR code
   - **Step 5**: Review and submit

4. **Track Status**:
   - Monitor progress through pipeline: Draft → Submitted → Under Scrutiny → EDS → Referred → MoM → Finalized
   - Respond to EDS (Environmental Data Sheet) requests
   - Receive notifications on updates

---

### 🔍 Scrutiny Team Workflow

**Goal**: Review applications and generate gist summaries

1. **Access Dashboard** → `/scrutiny`
   - View all submitted applications in table format

2. **Review Application**:
   - Click on any row to open side panel
   - View application details
   - Check uploaded documents
   - Use verification checklist

3. **Take Actions**:
   - **Auto-Generate Gist**: Click to create AI-powered summary
     - Review generated content
     - Accept or edit as needed
   
   - **Raise EDS**: Request additional environmental data
   
   - **Refer to Meeting**: Send to committee for review

4. **Filter & Search**:
   - Use search bar for specific applications
   - Filter by status (Under Scrutiny, EDS Raised, Verified, Referred)

---

### 📋 MoM Team Workflow

**Goal**: Generate and finalize Minutes of Meeting

1. **Access Dashboard** → `/mom`
   - View meeting calendar/schedule

2. **Manage Meetings**:
   - Click on meeting cards to view details
   - See applications referred for each meeting

3. **Generate MoM**:
   - Select application with available gist
   - Click "Generate MoM" button
   
4. **Edit Gist**:
   - Toggle edit mode
   - Use toolbar (Bold, Italic, Table)
   - Save draft
   - Preview changes

5. **Finalize Document**:
   - Click "Convert to MoM"
   - Review final document
   - Click "Lock & Finalize" to make permanent
   
6. **Export**:
   - Download as DOCX
   - Download as PDF

---

### 🛡️ Admin Workflow

**Goal**: System administration and analytics

1. **Access Dashboard** → `/admin`
   - Default view: Overview with statistics

2. **Monitor Analytics** (Overview Tab):
   - View clearance statistics
   - Analyze sector-wise data in bar chart
   - Review recent system activity

3. **Manage Users** (User Management Tab):
   - View all registered users
   - Add new users
   - Edit user details
   - Deactivate users
   - Assign roles

4. **Manage Templates** (Templates Tab):
   - Upload gist templates for different categories
   - Edit existing templates
   - Version control
   - Delete outdated templates
   
5. **Configure Sector Parameters**:
   - Set processing time limits
   - Configure sector-specific rules
   - Update fee structures

6. **System Settings**:
   - Email notification preferences
   - Payment gateway configuration
   - System parameters
   - Backup settings

## 🎯 Common Tasks

### Track an Application
1. Go to Home → Click "Track Your Proposal"
2. OR Login as Proponent → View "My Applications"
3. Click application ID to view details

### Submit New Application
1. Login as Proponent
2. Click "File New Application" (green button)
3. Complete all 5 steps
4. Submit with declaration

### Respond to EDS Request
1. Login as Proponent
2. Check Notifications panel
3. Click on EDS notification
4. Upload additional documents
5. Resubmit

### Generate Meeting Minutes
1. Login as MoM Team
2. View referred applications
3. Click "Generate MoM" for application with gist
4. Edit content if needed
5. Finalize and export

### View Analytics
1. Login as Admin
2. Default view shows statistics
3. Scroll down for sector-wise chart
4. View recent activity log

## 🎨 UI Elements Guide

### Status Badges
- **Blue** = Under Scrutiny, Submitted, Scheduled
- **Orange** = EDS Raised, Warning
- **Green** = Verified, Approved, MoM Generated, Active
- **Teal** = Referred
- **Gray** = Draft, Inactive

### Color Coding
- **Forest Green** (#1A5C1A) = Primary actions, environmental theme
- **Government Blue** (#003087) = Admin, official
- **Saffron** (#FF6B00) = Alerts, scrutiny actions
- **Purple** (#7B1FA2) = MoM team

### Icons Meaning
- 📄 FileText = Documents, applications
- ✓ Check = Verified, completed
- ⚠ Alert = Attention needed, EDS
- 🔔 Bell = Notifications
- 💳 Credit Card = Payment
- 📊 Chart = Analytics
- 👤 User = Profile, proponent
- 🔍 Search = Scrutiny, review
- 📅 Calendar = Meetings
- 🔒 Lock = Finalized, secure

## 🆘 Troubleshooting

### Issue: Cannot upload documents
- **Solution**: Check file format (PDF, DOC, DOCX only)
- **Solution**: Ensure file size < 10MB
- **Solution**: Clear browser cache

### Issue: Payment verification failed
- **Solution**: Wait for payment processing (2-3 minutes)
- **Solution**: Check UPI transaction ID
- **Solution**: Contact helpline if issue persists

### Issue: Cannot access certain screens
- **Solution**: Verify you're logged in with correct role
- **Solution**: Check role permissions
- **Solution**: Contact admin for access

### Issue: Page loading slowly
- **Solution**: Use latest Chrome/Firefox/Safari
- **Solution**: Clear browser cache
- **Solution**: Check internet connection
